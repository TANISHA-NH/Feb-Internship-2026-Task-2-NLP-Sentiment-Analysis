# Sentiment Analysis using NLP & ML

* **Internship Task:** GEN AI Internship – February 2026 (Task 2)

* **Objective:** Build an end-to-end Sentiment Analysis pipeline using NLP and ML models

* **Dataset:** IMDb Movie Reviews dataset (Kaggle)

* Contains text reviews labeled as **positive** or **negative**

* **NLP Preprocessing:**

  * Lowercasing
  * Removing punctuation & special characters
  * Stopword removal
  * Lemmatization

* **Feature Engineering:**

  * TF-IDF (Term Frequency–Inverse Document Frequency)

* **Models Used:**

  * Logistic Regression
  * Naive Bayes
  * Decision Tree

* **Evaluation Metrics:**

  * Accuracy
  * Precision
  * Recall
  * F1 Score

* **Result:**

  * Logistic Regression performed best
  * Naive Bayes showed good performance
  * Decision Tree had comparatively lower accuracy

* **Best Pipeline:**
  Preprocessing → TF-IDF → Logistic Regression
